# MagicDraw

## Overview
マウスで赤線をなぞると文字が現れる．

## Description
ウィンドウの赤線をなぞるようにドラッグアンドドロップすることで「タカタ」という文字が書かれる．
各線の位置は座標変換により取得されている．

## Usage
1. DxLib環境下で `magicdraw.cpp` をビルド(コンパイル+リンク)
1. できあがった実行ファイルを実行

## Install
```
$ git clone https://github.com/bonochof/magicdraw.git
```
